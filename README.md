
# Sistema de Orçamento para Marmoraria

Demo simples feito com:

- HTML
- CSS
- JavaScript

Funcionalidades:

- Cadastro de clientes
- Cadastro de materiais
- Cadastro de serviços
- Cálculo automático de área
- Orçamento automático
- Dashboard
- Armazenamento no navegador (LocalStorage)

Para usar:

1. Abra index.html
2. Ou publique no GitHub Pages
